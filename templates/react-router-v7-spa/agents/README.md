# React Router v7 SPA Template - Custom Agent

This directory contains a custom agent specialized for the React Router v7 SPA template, compatible with GitHub Copilot, Claude, Cursor, and other AI assistants.

## Agent

### React Router v7 SPA Specialist
**File**: `react-router-spa-specialist.agent.md`

Expert in React Router v7 single-page applications with client-side routing and modern React patterns.

**Use for:**
- Client-side routing patterns
- Code splitting and lazy loading
- TanStack Query integration
- URL state management
- Form handling with React Hook Form
- Performance optimization
- Progressive enhancement

## Using This Agent

### In Your IDE or AI Tool
1. Open this template in your IDE or AI tool (GitHub Copilot, Claude, Cursor, etc.)
2. Reference the `react-router-spa-specialist` agent
3. Ask questions or request help specific to React Router v7 SPA development

### On GitHub.com
1. Navigate to https://github.com/copilot/agents
2. Select this repository
3. Choose the `react-router-spa-specialist` agent
4. Provide your requirements
