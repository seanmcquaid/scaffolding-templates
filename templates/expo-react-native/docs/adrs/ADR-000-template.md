## Context

Describe the context and background that led to this decision. What problem are we trying to solve? What constraints or requirements exist? What is the current situation?

Include relevant technical, business, or organizational factors that influenced this decision.

## Decision

Clearly state the decision that was made. Be specific and unambiguous.

For example:
- "We will use React 19 as our UI library"
- "We will implement authentication using JWT tokens"
- "We will deploy to Vercel for production hosting"

## Rationale

Explain why this decision was made. What are the key benefits? How does it address the context and problems described above?

Include:
- Key advantages of this approach
- How it solves the problem at hand
- Alignment with project goals and constraints
- Technical or business benefits
