# TanStack Router SPA Template - Custom Agent

This directory contains a custom agent specialized for the TanStack Router SPA template, compatible with GitHub Copilot, Claude, Cursor, and other AI assistants.

## Agent

### TanStack Router SPA Specialist
**File**: `tanstack-router-spa-specialist.agent.md`

Expert in TanStack Router for type-safe single-page applications with file-based routing.

**Use for:**
- Type-safe routing with generated types
- Search parameter validation with Zod
- Route-based code splitting
- TanStack Query integration
- Type-safe navigation and links
- Route context management
- Performance optimization (prefetching, preloading)

## Using This Agent

### In Your IDE or AI Tool
1. Open this template in your IDE or AI tool (GitHub Copilot, Claude, Cursor, etc.)
2. Reference the `tanstack-router-spa-specialist` agent
3. Ask questions or request help specific to TanStack Router development

### On GitHub.com
1. Navigate to https://github.com/copilot/agents
2. Select this repository
3. Choose the `tanstack-router-spa-specialist` agent
4. Provide your requirements
